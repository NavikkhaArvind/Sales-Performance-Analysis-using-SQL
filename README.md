# Sales Performance Analysis using SQL

## Project Overview
This project analyzes sales data using MySQL to identify business insights such as top-performing regions, products, customers, and monthly sales trends.

## Tools Used
- MySQL
- SQL
- GitHub

## Dataset
- 200 sales records
- Includes customer, product, region, sales, profit, quantity, payment method, and order date.
